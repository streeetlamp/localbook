# LocalBook

Text styling functionality provided via [jquery-notebook](https://github.com/raphaelcruzeiro/jquery-notebook)

## How To: 
```bash
bundle install
npm install
grunt
```
Work in the `app` directory. 

Build with `grunt prod`
